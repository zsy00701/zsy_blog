# 欢迎来到我的博客

这是我的第一篇博客文章！这个博客使用 Next.js 构建，支持 Markdown 格式的文章。该博客框架由cursor生成。

## 功能特点

- 📝 支持 Markdown 格式的笔记和文章
- 🎨 现代化的 UI 设计
- 📱 响应式布局，支持移动端
- ⚡ 基于 Next.js，性能优秀
- 🚀 可以轻松部署到 Vercel、Netlify 等平台

## 如何使用

1. 在 `content/posts` 目录下创建 `.md` 文件
2. 在文件开头添加 frontmatter（元数据）
3. 使用 Markdown 语法编写内容
4. 保存后，文章会自动出现在首页

```javascript
function greet(name) {
  return `Hello, ${name}!`;
}

console.log(greet('World'));
```
