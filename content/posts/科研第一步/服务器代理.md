# 服务器代理

## SSH 远程端口转发 (SSH Remote Port Forwarding

> 利用 SSH 协议建立**反向加密隧道**，将远程服务器（Remote Host）的特定端口流量，通过安全通道（Secure Channel）转发至本地客户端（Local Host）的指定端口。
>
> **实践**：
>
> ~~~zsh
> 自己的电脑：
> ssh -R 7899:127.0.0.1:7899 用户名@服务器IP
> 服务器：
> git config --global http.proxy http://127.0.0.1:7899
> git config --global https.proxy http://127.0.0.1:7899
> ~~~
